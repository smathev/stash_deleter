### Behavioral Gotchas

1. **`is_missing: "rating"` works; `is_missing: "last_played_at"` does NOT** — "invalid is_missing field" error
2. **`last_played_at LESS_THAN <date>` includes null values** — scenes never played come through this filter

4. **IntCriterionInput has `value2`** — supports BETWEEN / NOT_BETWEEN modifiers
5. **TimestampCriterionInput uses ISO8601 strings** for both value and value2
6. **VideoFile.size type is `Int64`** — Python must handle large integers (>2^31 for big files)
